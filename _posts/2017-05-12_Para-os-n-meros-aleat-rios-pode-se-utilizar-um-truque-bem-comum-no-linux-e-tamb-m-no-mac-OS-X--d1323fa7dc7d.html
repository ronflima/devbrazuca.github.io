Para os números aleatórios pode-se utilizar um truque bem comum no linux e também no mac OS X:

int i;
uint64_t value = 0;
FILE *fp = fopen("/dev/random", "r");
if (!fp) {
    perror("randgetter");
    exit(-1);
}
for (i=0; i<sizeof(value); i++) {
    value <<= 8;
    value |= fgetc(fp);
}
fclose(fp);

Este gerador é bem confiável. Dá para usar de forma geral.
